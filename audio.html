Lossless Audio:

  • MPEG-4 SLS
  • Audio Interchange File Format
  • Windows Media Audio Lossless

Why us Lossless File Formats

Using lossless audio format gives you an exact bit-for-bit copy of the original music.
You are able to convert a lossless file into any format and have the highest quality that format can support.

Lossy Audio Files:

  • OGG
  • MP4
  • MP3

How does Lossy compression work

When you convert an audio file to a lossy format such as MP4, the machine analyses all the
frequencies of the audio that you have formatted, then it discards frequencies that the human
cannot hear, but for very low frequency sounds it is usually filtered out having
a minor impact on what the audio sounds like.

Why use lossy file formats

You should use lossy format files because it uses up less space and it
brings out the best sounding audio when you listen to it
